## 🎮 遊戲介紹

**Z勇者塔** 是一款復古像素風的**橫版塔防攀登遊戲**。

在由神祕代碼構築的無盡高塔中，你將沿著**Z字階梯**不斷向上挑戰。  
跳躍、攻擊、踩怪、拾取裝備，一路從第0層殺到塔頂！

> **「Z字塔頂，只有真正的勇者才能抵達！」**

- 無盡模式，挑戰你的極限樓層
